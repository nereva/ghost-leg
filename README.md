# 🪜 네온 시크릿 사다리 게임 (Ghost Leg)

> 주말 야구 관람자 선정부터 회식 벌칙 정하기까지!  
> 화려한 네온 애니메이션과 비밀 장막이 펼쳐지는 프리미엄 사다리 게임 🎮⚾

## ✨ 주요 기능

### 🎯 자유로운 게임 설정
- **커스텀 게임 제목** — 야구 관람, 회식 벌칙, 선물 교환 등 원하는 주제로 자유롭게 변경
- **인원 조절** — 최소 2명 ~ 최대 10명까지 동적으로 추가/삭제
- **상품 & 당첨 설정** — 1등, 2등, 꽝 등 다양한 상품과 당첨 여부를 자유롭게 지정

### 🎚️ 사다리 복잡도 선택
- **단순 모드** — 가로선이 적어 직관적인 경로
- **복잡 모드** — 가로선이 촘촘하게 배치되어 결과 예측이 훨씬 어려움

### 🔮 비밀 장막 (블라인드) 시스템
- 사다리 중앙부가 **Glassmorphism 스타일의 불투명 장막**으로 완벽히 가려짐
- 사전에 눈으로 경로를 추적하는 것이 **100% 불가능**
- 사다리 타기가 시작되면 네온 빛이 장막을 뚫고 경로를 드러냄

### 🌈 인원별 고유 네온 컬러
- 각 참가자마다 **서로 다른 네온 색상**(사이언, 핑크, 그린, 오렌지 등)이 지정
- 사다리 완주 후에도 고유한 색상 트레일이 남아 **한눈에 경로 추적 가능**

### ⚡ 스마트 애니메이션 제어
- **순차 진행** — 왼쪽에서 오른쪽으로 한 명씩 사다리를 타고 내려감
- **쾌속 모드** — 모든 당첨 상품의 주인이 정해지면, 나머지 인원은 초고속으로 하강

### 🎉 당첨 축하 세레머니
- 화면 전체에 쏟아지는 **오색 폭죽(Confetti) 이펙트**
- Web Audio API로 합성된 **트럼펫 팡파레 효과음** 🎺
- 당첨자를 빛내는 골드 하이라이트 결과 카드

### 📱 반응형 디자인
- PC, 태블릿, 모바일 등 **모든 디바이스에 최적화**
- 카카오톡 인앱 브라우저에서도 깔끔하게 작동

## 🛠️ 기술 스택

| 구분 | 기술 |
|------|------|
| **마크업** | HTML5 (Semantic) |
| **스타일링** | Vanilla CSS (Glassmorphism, Neon Theme, Responsive) |
| **로직** | Vanilla JavaScript (Canvas API, Web Audio API) |
| **폭죽 효과** | [canvas-confetti](https://github.com/catdad/canvas-confetti) |
| **아이콘** | [Font Awesome 6](https://fontawesome.com/) |
| **폰트** | [Google Fonts](https://fonts.google.com/) — Outfit, Noto Sans KR, Orbit |

> 프레임워크 없이 **순수 HTML/CSS/JS**만으로 구현되어 별도의 빌드 과정이 필요 없습니다.

## 🎮 사용법

1. **게임 제목** 입력 (예: "주말 야구 관람자 선정! ⚾")
2. **인원 수** 조절 후 참가자 이름과 상품 입력
3. 당첨 상품은 **토글 스위치**를 켜서 지정
4. **사다리 복잡도** 선택 (단순 / 복잡)
5. **사다리 만들기** 클릭 → 게임판 생성
6. **사다리 타기 시작!** 클릭 → 네온 애니메이션 감상 🌟
7. 당첨자 발표와 함께 폭죽 세레머니! 🎊
